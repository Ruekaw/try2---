# Q3（K=3）职业舞者与明星特征影响：结论草稿（可直接写入报告）

> 目的：量化职业舞者（Pro）与明星特征（年龄、行业等）对比赛表现的影响，并比较这些因素对“评委打分”和“粉丝投票（Q1 反推）”的影响方式是否相同。

## 1. 数据口径与建模口径（写作时建议保留 3 句话）

- **样本窗口（缓解生存偏差）**：仅使用每赛季前 $K=3$ 周且当周确实参赛（`is_competing_week==True`）的记录。K 的选择基于覆盖率与生存率：在候选 $K\in\{3,4,5\}$ 中，$K=3$ 的赛季覆盖率为 1.0，平均生存率约 0.854；而 $K=5$ 生存率降至约 0.676（详见 `k_selection.json`）。
- **可比因变量（同周份额的 centered-log）**：在同一 `(season, week)` 组内，将评委份额与粉丝份额都转换为 $y=\log(share+\varepsilon)-\text{mean}_{(season,week)}\log(share+\varepsilon)$，从而消除不同赛季/周人数变化带来的尺度不可比。
- **两套结构一致的分层回归**：分别对 `y_judge`（评委侧）与 `y_fan`（粉丝侧）拟合同一结构：
  
  $$y \sim age_c + log\_week + industry + (1|ballroom\_partner) + (1|season).$$

## 2. 量化影响（How much impact?）——固定效应（年龄/行业）

### 2.1 年龄（Age）

- **评委侧（judge）**：`age_c` 为**显著负向**，后验均值约 **-0.005**，95%HDI 约 **[-0.006, -0.005]**。
- **粉丝侧（fan）**：`age_c` 同样为负向但**更弱**，后验均值约 **-0.003**，95%HDI 约 **[-0.005, -0.001]**。

解释口径（建议直接用）：
- 在“同赛季同周的相对份额(log-share)”口径下，年龄越大，早期周的相对表现越吃亏；且评委对年龄的“惩罚”强度大于粉丝。

### 2.2 行业（Industry）

- **评委侧（judge）**：多数行业相对基准行业呈现稳定负向（例如 `Model`、`Politician`、`TV/Host`、`Athlete` 等系数显著为负；`Musician` 接近 0）。
- **粉丝侧（fan）**：行业效应整体更不稳定（区间更宽），但 `Model` 仍显著为负（均值约 **-0.261**，95%HDI 约 **[-0.432, -0.103]**），且负向幅度大于评委侧（评委 `Model` 约 **-0.117**）。

解释口径（建议直接用）：
- 行业标签对“评委评分份额”存在更稳定的区分；对“粉丝投票份额”的区分在不确定性更大，但部分行业（如 Model）在两侧均表现为显著劣势，并且粉丝侧差异可能更强。

### 2.3 周次趋势（log_week）

- 评委侧与粉丝侧的 `log_week` 均接近 0 且 95%HDI 跨 0，说明在仅取前 3 周的窗口里，“越往后越占优/越吃亏”的系统趋势不明显。

## 3. 差异性比较（Same way?）——粉丝 vs 评委差值

我们对共同固定效应参数计算差值 $\Delta\beta = \beta_{fan}-\beta_{judge}$。

- **年龄差异（核心结论）**：$\Delta\beta_{age\_c}\approx +0.002$，95%HDI 约 **[0.0, 0.004]**。
  - 结论：粉丝侧对年龄的负效应显著**更弱**（差值为正，意味着粉丝比评委“更少惩罚年龄”）。

- **行业差异（概括结论）**：行业相关差值区间更宽，提示“粉丝与评委对行业标签的偏好/惩罚可能不同”，但不同行业的显著性存在差异，建议在正文中用 1–2 个代表行业（如 Model、TV/Host）做重点对比陈述。

## 4. 不确定性（把 Q1 的不确定性传递到 Q3）

为避免只用 Q1 均值导致过度自信，我们基于 `Q1_data_expo/*.npz` 的粉丝份额后验样本做了多次拟合汇总：

- 在跨样本集的差值汇总中，`age_c` 的 **p(Δ>0)≈0.96**（见 `fan_sample_delta_quantiles_k3.csv`），说明“粉丝更不惩罚年龄”这一差异对 Q1 不确定性具有**较强稳健性**。
- `log_week` 的差值在样本集层面接近 0（p 值约 0.5），与“前 3 周窗口周次趋势不明显”的结论一致。

## 5. 关于职业舞者（Pro）效应：当前 K=3 结果的限制说明（务必写上）

- 本次输出中 `judge_pro_effects_k3.csv` 与 `fan_pro_effects_k3.csv` 为空，因此无法基于现成文件给出：
  - “各 Pro 的随机效应排名/Top-Bottom”；
  - “同一 Pro 在评委侧与粉丝侧效应是否一致”的相关系数（`pro_effect_correlation_k3.json` 的 `n_pairs=0`）。

建议写作口径（不影响交稿）：
- 报告中仍可将 Pro 作为模型结构的一部分（用于吸收配对差异），但**在 K=3 的现有导出结果下**，我们对 Pro 的“定量排名结论”暂不展开；本问核心差异结论主要由年龄与行业等固定效应支撑。

## 6. 一句话总结（放在 Q3 开头/结尾都合适）

在仅取每赛季前 3 周以降低生存偏差的框架下，年龄对相对表现呈稳定负效应；且粉丝侧对年龄的惩罚显著弱于评委侧（差值稳健）。行业对评委侧的区分更稳定，而粉丝侧行业差异不确定性更大但个别行业（如 Model）仍表现为显著负向。
